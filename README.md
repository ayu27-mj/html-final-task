# チーム開発演習(Git/GitHub)　最終課題　レベル1
## サイトについて
 htmlファイル10個、cssファイル2個、JSファイル2個、imagesフォルダで構成されています。
 このサイトは元のアニメレビューサイトを仮面ライダーやスーパー戦隊のレビューサイトに改造したものです。
## 変更点
- トップページの注目の作品の項目を削除し、カルーセルを追加 
- ほぼすべてのページにおいて、ナビゲーションバーがずっと画面上部に固定されるように変更
- ランキングページの「今期」の項目を削除し、ボタンを見やすいように黒い枠を追加
- マイリストページのボタンを見やすいように黒い枠を追加
- FAQのページを作成
- 作品詳細ページのカルーセルを削除し、代わりに動画の埋め込みを追加
- 作品詳細ぺージのキャラクターの要素を、アコーディオンからタブコンテンツへ変更
- 作品詳細ページの統計情報の「視聴済」などの要素にアイコンを追加
## 工夫した点
- 見やすさを考えて、ランキングページやマイリストページのボタンに黒い枠を追加したり、間にmarginをかけた。
- 作品詳細ページのボタンも同じように変更しようと思ったが、そのままの方が見やすかったのでそのままにした。
- FAQページで余計な質問まで表示されると、自分が知りたい質問が探しにくいと思い要素をアコーディオンにした。
- トップページのカルーセルをスマホで見たときに、説明文もあると見ずらくなってしまうので、スマホの場合は非表示になるようにした。
## 苦労した点
- トップページのカルーセルをレスポンシブ対応にするのが大変だった。
- 特に画像の処理でとても苦労した。
